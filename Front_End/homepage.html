Objetivo do Projeto

Com este projeto, o principal objetivo era criar uma plataforma onde fosse possível gerir e consultar toda a informação relevante sobre os eventos realizados na pista do Estoril. Desde os eventos passados, aos recordes da pista, aos pilotos e equipas que participaram, tudo está organizado e acessível de forma simples e intuitiva. Usando os conhecimentos obtidos nas aulas da disciplina de Fundamentos de Base de Dados,criámos uma UI responsiva que usa um conjunto de queries e funções em SQL para interagir com a base de dados criando a experiência de utilizador final.
